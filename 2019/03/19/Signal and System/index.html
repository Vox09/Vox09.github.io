System

Basic System Characterization

  1. Memoryless: instantaneous
  2. Causality: does not depend on future
  3. Stability: BIBO(Bounded-Input Bounded Output)
  4. Invertibility: distinct inputs lead to distinct outputs
  5. Time-invariant
  6. Linearity

Time Invariance

Shifted input shifted output.

Operation of time does not depends on time.

$ y(t) = \int_{t-3}^{t+2} x(\tau)d\tau$ YES : 3 time before and 2 later
$ y(t) = \int_{0}^{t} x(\tau)d\tau$ NO: has a beginning point 0

Linearity

  1. Additive property: x1(t)→x2(t)=y1(t)→y2(t)x_1(t) \to x_2(t) = y_1(t) \to y_2(t)x1​(t)→x2​(t)=y1​(t)→y2​(t)
  2. Homogeneity (scaling): ax1(t)→ay1(t)ax_1(t) \to ay_1(t)ax1​(t)→ay1​(t)

Elementary Linear Operation:

  1. Differentiation and integration: x(k)(t)x^{(k)}(t)x(k)(t) ( LTI! )
  2. Time shifting:x(t−τ)x(t-\tau)x(t−τ) ( LTI! )
  3. Multiply by a function: g(t)x(t)g(t)x(t)g(t)x(t) ( not TI unless g(t)g(t)g(t) is constant)
  4. time reversal, time scaling ( not TI because there is a origin)

Denoted by h(t)h(t)h(t) or h[n]h[n]h[n]

If the output of a system is given by convolving the input with a h[n]h[n]h[n], then the system is LTI! Convolution is LTI
