Signal and System

Posted on 2019-03-19 Edited on 2020-03-24 In Signal is Everything
HKUST ELEC2100 notes

Signals

  • What is signal?
    A signal is some measurable quantity that varies over time and/or space.
  • What are systems?
    We define system to be anything that takes input signals and produces output signals.

Basic Characterization and Manipulation

Manipulation

  1. multiplying signal by constant
  2. adding/multiplying signals together
    • Multiplying two straight line segments produces a quadratic curve
  3. transformation of the independent variable
    1. time shifting $x(t)\to x(t-t_0)$
      Minus = Left = Delay
    2. time reflection/reversal $x(t)\to x(-t)$
    3. time scaling $x(t)\to x(at)$

Combination of shifting and time scaling
Shift first and scale is easier. First scale requires we also scale the shift.

Characterization

  1. Even signal $x(-t)=-x(t)$
    Odd signal $x(-t)=x(t)$
    Any signal can be viewed as the sum of an even part and an odd part $x_{even}={x(t)+x(-t)\over 2}\quad x_{odd}={x(t)-x(-t)\over 2}$
  2. infinite duration & finite duration
    • A finite duration DT signal is a sequence of N complex numbers. We can treat it as a complex N-vector.
  3. right-sided & left-sided
    right-sided $u(t-c)$ left-sided $u(-t-c)$

Fundamental Signals

CT

Complex Exponential

Where $e^{\sigma t}$ is purely real. $\sigma>0$ amplitude grows, $\sigma<0$ amplitude decays.
$e^{j\omega t}$ is purely imaginary and called Complex Sinusoids

Complex Sinusoids

$e^{j\omega t} = cos\omega t +j sin\omega t$

Unit Function

$ u(t) = 1$ when $t\ge 0$
$ u(t) = 0$ when $t\lt 0$

Impulse Signal

Denoted by $\delta(t)$

  1. Zero everywhere else, infinite at $t=0$
  2. Unit total area $\int_{-\infty}^{\infty} \delta(t)dt = 1$
  3. Integral is the unit step $u(t) = \int_{-\infty}^{\infty} \delta(\tau)d\tau$
  4. Scaled impulse k$\delta(t)$ has total area k. $\int_{-\infty}^{\infty} k\delta(t)dt = k\int_{-\infty}^{\infty}\delta(t)dt = k $
  5. Sampling property
    $x(t)\delta(t-t_0) = x(t_0)\delta(t-t_0)$ a scaled delta function
  6. Sifting property
    $\int_{-\infty}^{\infty}x(t)\delta(t-t_0)dt = x(t_0)$ a value at t~0~
  7. Infinite energy. Therefore $\delta(t)$ is an idealization that cannot physically exist.
  8. Time Scaling corresponds to Magnitude Scaling $\delta(\alpha t) = {\delta(t)\over \left|{\alpha}\right|}$

DT

Real Exponential

$ \alpha^n = e^{an} $ where $\alpha = e^a$

$\alpha>1$ growing exponential
$0<\alpha<1$ decaying exponential
$-1<\alpha<0$ decaying oscillating
$\alpha<-1$ growing oscillating

Real Sinusoids

$cos(\omega n) = cos(2\pi fn)$

  1. Periodic only if its ordinary frequency $f$ is rational number
  2. $f$ and $f\pm m(integer)$ are the same frequency.
    $\omega$ and $\omega\pm m2\pi$ are the same frequency
  3. The fastest rate at which a DT signal can oscillate is at an ordinary frequency of $f={1\over2}$ or angular frequency of $\omega = \pi$

    Complex Exponential

$z^n$ where $z = e^s = e^{\sigma + j\omega} = e^\sigma e^{j\omega}$

$z^n = \left|z\right|^n e^{j\omega n}$
$\left|z\right|^n$is real exponential $ e^{j\omega n}$ is complex sinusoid

Complex Sinusoid

$x[n] = e^{j\omega n} = cos \omega n + j sin \omega n$

  1. Periodic iff $\omega = {m2\pi \over N}$
  2. $e^{j\omega n} = e^{j(\omega +2k\pi)n}$ frequency is periodic for DT signal, and the highest angular frequency is $\pi$

Unit Step

$u[n] = 1$ when $n\ge 0$
$u[n] = 0$ when $n\lt 0$

Unit Impulse

$\delta[0] = 1$ other place 0.
Property same as CT.

Relations with Unit step

  1. the first difference between two unit step signals.
    $\delta[n] = u[n]-u[n-1]$
  2. unit step can be expressed as a sum of impulse by two ways
    1. $u[n]\sum_{m=-\infty} ^n \delta[m]$
    2. $u[n]\sum_{k=0}^\infty \delta[n-k]$
  3. the sampling property for $\delta[n]$
    when multiply $x[n]$ and $\delta[n]$, only one sampled value, not a continuous function anymore.

Complex Frequency

CT and DT Complex Frequency $z = e^s = e^{\sigma+ j\omega}$

DT CT
Growing $ z >1$ $\sigma>0$
Decaying $ z <1$ $\sigma<0$
Oscillatory frequency $\ang z$ $\omega$

System

Basic System Characterization

  1. Memoryless: instantaneous
  2. Causality: does not depend on future
  3. Stability: BIBO(Bounded-Input Bounded Output)
  4. Invertibility: distinct inputs lead to distinct outputs
  5. Time-invariant
  6. Linearity

Time Invariance

Shifted input shifted output.

Operation of time does not depends on time.

$ y(t) = \int_{t-3}^{t+2} x(\tau)d\tau$ YES : 3 time before and 2 later
$ y(t) = \int_{0}^{t} x(\tau)d\tau$ NO: has a beginning point 0

Linearity

superposition property can be decomposed into two

  1. Additive property: $x_1(t) \to x_2(t) = y_1(t) \to y_2(t)$
  2. Homogeneity (scaling): $ax_1(t) \to ay_1(t)$

LTI must satisfy zero-input/zero-output

Elementary Linear Operation:

  1. Differentiation and integration: $x^{(k)}(t)$ ( LTI! )
  2. Time shifting:$x(t-\tau)$ ( LTI! )
  3. Multiply by a function: $g(t)x(t)$ (not TI unless $g(t)$ is constant)
  4. time reversal, time scaling ( not TI because there is a origin)

Impulse Response

Denoted by $h(t)$ or $h[n]$, is the output of $\delta(t)/\delta[t]$ in a LTI system.

If the output of a system is given by convolving the input with a $h[n]$, then the system is LTI! Convolution is LTI

Typical Examples:

  1. $h[n] = \delta[n]$ identity function. Do nothing.
  2. $h[n] = \delta[n-m]$ delay system. Delay the input by m.
  3. $h[n] = \delta[n]+0.3\delta[n-4]$ echo system. With echo volume 0.3.
  4. $h[n] = u[n]$ first sum system. $y[n] = \sum_{-\infty}^n x[k]$
  5. $h[n] = {1\over 3} (u[n] - u[n-3]) = {1\over 3}(h[n] + h[h-1] + h[n-2])$ window smoother.
  6. $h[n] = \delta[n] -\delta[n-1]$ first difference system.

LTI Systems

Linearity ensure that
If $\delta[n-k]\to h_k[n]$ we can imply
$x[n] = \sum_{k=-\infty}^{\infty} x[k]\delta[n-k]\to y[n] = \sum_{k=-\infty}^{\infty} x[k]h_k[n-k]$

Time Invariance ensure that
In fact $h_k[n]=h[n-k]$
$x[n] = \sum_{k=-\infty}^{\infty} x[k]\delta[n-k]\quad y[n] = \sum_{k=-\infty}^{\infty} x[k]h[n-k]$

For CT case
$x(t) = \int_{-\infty}^{\infty}x(\tau)\delta(t-\tau)d\tau\quad y(t) = \int_{-\infty}^{\infty}x(\tau)h(t-\tau)d\tau$
we use $$ to denote it.
$y(t)=x(t)h(t)$

Convolution/LTI properties:

  1. commutative $x(t) \ast h(t) = h(t) \ast x(t)$
  2. distributive $x(t) \ast\{h_1(t) + h_2(t)\} = x(t)\ast h_1(t) +x(t) \ast h_2(t)$
  3. associative$\{x(t)\ast h_1(t)\}\ast h_2(t) = x(t)\{h_1(t)h_2(t)\}$

Charactering LTI system by Impulse Responses:

  1. Memoryless <=> $h[n] = 0 $ for $n\ne0$
  2. Causality <=> $h[n] =0$ for $n\lt0$
  3. Stability <=>impulse response is absolute integrable/summable
    $\int_{-\infty}^{\infty}\left|h(t)\right|dt \lt \infty$ or $\sum_{n=-\infty}^{\infty}\left|h(t)\right|dt \lt \infty$

If the impulse response of the system is a unit step function, the system is a integral/first sum system.

Conceptually computing

For the DT convolution $y[n]=x[n]*h[n]=\sum_{k=-\infty}^\infty x[k]h[n-k]$

  1. draw the original $x[k]$ and $h[k]$ with variable $k$
  2. flip $h[k]$ to get $h[-k]$, the convolution kernel
  3. shift $h[-k]$ right by $n$ to obtain $h[n-k]$
  4. Multiple $x[k]$ by $h[n-k]$ and sum over all results

System Function and Frequency Response

System Function

$z^n $/$ e^{sn}$ is an eigenfunction of DT/CT LTI systems. $H(z)$ or $H(s)$ is the eigenvalue that depends on the complex frequency $z$. They are also called system function of CT/DT LTI systems.

$H(s)$ is a Laplace Transform of the CT impulse response $h(t)$
$H(z)$ is a z-transform of the DT impulse response $h[n]$

Frequency Response

For LTI System, the eigenvalue at different oscillation frequency $\omega$ is called frequency response.

$H(s)e^{st} = H(j\omega)e^{j\omega t}$ when $s=\sigma + j\omega$ and $\sigma=0$
where $H(j\omega)$ is the cross-section of $H(s)$ along $j\omega$ axis.

$H(z)z^{n} = H(e^{j\omega})e^{j\omega n}$ when $|z|= 1$ and $\ang z=\omega$
where $H(e^{j\omega})$is the value of system function $H(z)$ along the unit circle $z=e^{j\omega}$

$H(j\omega)=H(s)$ given that $s=j\omega$
$H(e^{j\omega})=H(z)$ given that $|z|=1;z=e^{j\omega}$
